# LogBlade 0.1.0 Beta 1

This is the first public beta of LogBlade, a portable Win32 log viewer focused on opening and navigating large text logs quickly.

## Included

- Progressive, viewport-first loading.
- Multi-stage text and regular-expression filtering.
- Configurable display parsers and highlighting rules.
- Live file growth tracking.
- File, command-line, and clipboard input.
- A self-contained NativeAOT executable for Windows x64.

## Install

Download `LogBlade-0.1.0-beta.1-win-x64.zip`, verify its hash against `SHA256SUMS.txt`, extract it, and run `LogBlade.exe`.

This beta is portable: it does not include an installer, automatic updates, or file associations. It is also not digitally signed, so Windows SmartScreen may show an unrecognized-app warning.

# LogBlade 0.1.0 Beta 1 — Português

Esta é a primeira beta pública do LogBlade, um visualizador Win32 portátil voltado à abertura e navegação rápida de logs de texto grandes.

## Incluído

- Carregamento progressivo priorizando a área visível.
- Filtros de texto e expressão regular em múltiplas etapas.
- Parsers de exibição e regras de destaque configuráveis.
- Acompanhamento de arquivos que continuam crescendo.
- Entrada por arquivo, linha de comando e área de transferência.
- Executável NativeAOT autocontido para Windows x64.

## Instalação

Baixe `LogBlade-0.1.0-beta.1-win-x64.zip`, confira o hash usando `SHA256SUMS.txt`, extraia o conteúdo e execute `LogBlade.exe`.

Esta beta é portátil: ela ainda não inclui instalador, atualização automática ou associação de arquivos. O executável também não possui assinatura digital, portanto o Windows SmartScreen pode exibir um aviso de aplicativo não reconhecido.
